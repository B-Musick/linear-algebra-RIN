MATRIX CALCULATOR

How many matrices?

  • Addition(+):

    - '0+1' would add matrix 0 and matrix 1.

  • Subtraction(-):

    - '0-1' would subtract matrix 0 and matrix 1.

  • Multiplication(*):

    - '0*1' would multiply matrix 0 by matrix 1.

  • Transpose(t):

    - '0t' would transpose matrix 0.

  • Trace(e):

    - '0e' would give the trace of matrix 0.
    - Trace is the sum of all the values in the diagonal of the matrix thus the matrix must be size nxn.
    - The values in the diagonal are also known as eigenvalues, hence the use of the letter 'e' for the forumula submission.
